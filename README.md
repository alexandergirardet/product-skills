# Product Skills

Claude Code plugin marketplace for product management workflows.

## Install

```bash
# Add the marketplace
/plugin marketplace add alexandergirardet/product-skills

# Install the PRD workflow (22 PM skills)
/plugin install prd-development@alavida-pm-skills
```

## Plugins

### prd-development

Structured PRD development workflow with 22 composable PM skills — from problem statement through user stories.

Includes: problem-statement, proto-persona, jobs-to-be-done, user-story, user-story-splitting, user-story-mapping, epic-hypothesis, epic-breakdown-advisor, opportunity-solution-tree, discovery-process, and more.
